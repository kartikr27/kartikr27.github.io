Cassini

Robot for 2023 FRC Charged Up season

Robot Codebase

The robot code base consisted of two main components. The motor controller side in Java, which would store the setpoints, PID loops, and controller mappings and the more complex side of ROS in Python that we ran on a co-processor in Linux which would do path-planning for the robot. We would send values from the ROS side to the Java side through an ethernet service called NetworkTables. Click the following links for the repositories!

Autonomous

During the first 15 seconds of the match robots are autonomous We utilized ROS (Robot Operating System) to create a custom path editing website. In this website the user can draw splines and define points for the robot to follow. Based on the drawn splines the algorithm would calculate target velocites at several points during each path for the robot to follow.

Robot breakdown

For the 2023 FRC Charged Up competition we built a robot to pick up cones and cubes and place them in an area called the grid to score points.

This robot consisted of 4 swerve drive modules that allow the robot to translate and rotate simultaniasly, carbon fiber extending arm with a rotating wrist, and a variable angle shooter. Along with these the complex electronics and motor controllers were crucial to competiting at the highest level at the Worlds stage.

After qualification matches we seeded 6th in one of the most competitive divisons at FRC Worlds Competition that year and captained the 5th alliance. We placed 3rd in our divison after playoffs.
